

1) Aktivirati virtualno okruzenje:
	Preko CMD upisati (za pokretanje skripte mora se biti u Admin modu):
	C:\> VRPvirtual\Scripts\activate.bat

2) Pokrenuti server preko virtualnog okruzenja:
	(VRPvirtual) C:\> python path\to\manage.py runserver
	Napomena: manage.py se nalazi unutar foldera s django aplikacijom.
